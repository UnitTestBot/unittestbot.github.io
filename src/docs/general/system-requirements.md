# System Requirements and Supported Systems
## Client Side
The plugin is available for VS Code (1.50+) and its extensions: Huawei VS Code and WeCode. 

## Server Side
**Operating Systems**

Linux : Ubuntu 14, 16, 18, 20, OpenSUSE

Windows : WSL 1, 2 with Ubuntu 14.04+
